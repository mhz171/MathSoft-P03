# پروژه سوم درس نرم‌افزار ریاضی

این مخزن شامل کدهای پیاده‌سازی شده برای پروژه سوم درس نرم‌افزار ریاضی است. در این پروژه، مفاهیم مختلف محاسبات علمی، درونیابی و تحلیل داده‌ها با استفاده از پایتون پیاده‌سازی شده‌اند.

## 📌 ساختار و بخش‌های پروژه

این پروژه از سه بخش (فایل‌های Jupyter Notebook) تشکیل شده است:

* **بخش اول (`Q01.ipynb`): درونیابی لاگرانژ و اسپلین**
  * محاسبه ضابطه دقیق و رسم نمودار درونیابی چندجمله‌ای لاگرانژ و اسپلین مکعبی (Cubic Spline) روی مجموعه‌ای از نقاط مشخص و مقایسه آن‌ها.
  
* **بخش دوم (`Q02.ipynb`): تحلیل زمان اجرای الگوریتم‌ها**
  * فراخوانی داده‌های زمان اجرا از فایل اکسل (`s02.xlsx`) و رسم نمودارهای میله‌ای، خطی و جعبه‌ای برای تحلیل عملکرد سه الگوریتم مختلف.
  
* **بخش سوم (`Q03.ipynb`): انتگرال‌گیری عددی**
  * محاسبه انتگرال تابع $f(x) = x^{2^x}$ در بازه (۰,۱) با استفاده از سه روش: قاعده ذوزنقه‌ها، قاعده سیمپسون و روش گوس-لژاندر و تحلیل خطای هر روش.

## 🛠️ پیش‌نیازها و نصب

برای اجرای صحیح کدهای این پروژه، به زبان برنامه‌نویسی **Python 3** و کتابخانه‌های علمی زیر نیاز دارید:

- `numpy`
- `scipy`
- `pandas`
- `matplotlib`
- `openpyxl` (برای خواندن فایل‌های اکسل)

جهت نصب تمامی کتابخانه‌های مورد نیاز، دستور زیر را در ترمینال یا محیط مجازی خود اجرا کنید:

```bash
pip install numpy scipy pandas matplotlib openpyxl
```
